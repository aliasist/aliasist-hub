# Aliasist Hub

![Aliasist banner](https://raw.githubusercontent.com/aliasist/aliasistabductor/master/images/aliasist_banner_orbit.png)

Aliasist uses this repo as the public app hub.

It provides:

- a launch point for the suite
- simple project cards and status checks
- a small public interface
- the entry surface for the Aliasist products

This repo stays focused on discovery and navigation.
